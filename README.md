# To-Do List App

A simple React-based To-Do List application.

## Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Strike-through completed items

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React
- Vite
- CSS
